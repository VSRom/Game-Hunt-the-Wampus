	Hunt the Wampus — это классическая текстовая игра, изобретённая Грегори Йобом (Gregory Yob) в 1973 году.
Игрок исследует пещерный лабиринт из 20 комнат, пытаясь убить хитрого монстра Вампуса с помощью лука и стрел, избегая при этом смертельных опасностей.

Геймплей:

	Вы в комнате: 7
	Соседние комнаты: 9 0 5
	Я слышу летучую мышь...
	Я чувствую ветерок...

	Что вы будете делать?
	> m9          # перейти в комнату 9
	> s9-0-5      # выстрелить через комнаты 9→0→5

Правила: 

	Летучая мышь - Переносит игрока в случайную комнату "I hear a bat"
	Бездонная яма - Мгновенная смерть при входе 		"I feel the breeze"
	Вампус - Съедает игрока при встрече 				"I smell Wampus"
	Стрела - Может убить Вампуса на расстоянии - Пролетает через 3 комнаты

	Победа: Убить Вампуса стрелой.
	Поражение: Упасть в яму, встретиться с Вампусом или потратить все стрелы.

Технические преимущества:

    Процедурная генерация уровня — каждый запуск создаёт уникальную связную пещеру
    Валидация данных — гарантии корректности графа, размещения опасностей и спавна игрока
    Модульная архитектура — лямбда-выражения для инкапсуляции логики проверок
    Современный C++ — использование возможностей стандартов C++11/14
	
	Быстрый старт:
	Требования:

    Компилятор с поддержкой C++11 или новее (g++, clang++, MSVC)
    Стандартная библиотека C++
	
	# Linux / macOS
	g++ -std=c++14 -O2 -o wampus Game.cpp

	# Windows (MinGW)
	g++ -std=c++14 -O2 -o wampus.exe Game.cpp

	# Windows (MSVC)
	cl /std:c++14 /O2 Game.cpp
	
Запуск: 

	./wampus
